## 🎯 Project Overview

An intelligent healthcare system that leverages AI to detect skin diseases from images, provide health recommendations, and connect users to language-matched doctors. This project combines deep learning, computer vision, and healthcare informatics to support early screening and guidance.

## ✨ Key Features

- 🔍 **AI-Powered Detection**: Classifies skin conditions into 22 different disease categories
- 📊 **Confidence Scoring**: Provides probability scores for predictions
- 🩺 **Doctor Consultation Matching**: Recommends doctors based on condition and preferred language
- 💊 **Health Guidance**: Offers condition-specific health recommendations
- 📝 **Feedback System**: Collects user feedback to improve system accuracy
- 🌐 **Web Interface**: User-friendly Streamlit-based web application

## 🚀 Quick Start

### Prerequisites
- Python 3.8 or higher
- 4GB RAM minimum (8GB recommended)
- Trained model files in the root `models/` directory

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd AI-Based-Intelligent-System-for-Skin-Disease-Detection-and-Healthcare-Recom_Feb_Batch-8_2026
   ```

2. **Create and activate virtual environment**
   ```bash
   # Windows
   python -m venv .venv
   .venv\Scripts\activate

   # Linux/Mac
   python3 -m venv .venv
   source .venv/bin/activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r SkinDisease/requirements.txt
   ```

4. **Run the application**
   ```bash
   cd SkinDisease
   python -m streamlit run app.py
   ```

5. **Access the application**
   - Open your browser and navigate to: `http://localhost:8501`

## 🔬 Technology Stack

### Machine Learning & AI
- **Framework**: TensorFlow 2.10+, Keras
- **Architecture**: MobileNetV2 (Transfer Learning)
- **Optimizer**: Adam with adaptive learning rate
- **Regularization**: Batch Normalization, Dropout

### Web Application
- **Framework**: Streamlit
- **Visualization**: Plotly, Matplotlib, Seaborn
- **Data Processing**: NumPy, Pandas
- **Image Processing**: PIL (Pillow)

### Model Training
- **Data Augmentation**: ImageDataGenerator
- **Callbacks**: EarlyStopping, ReduceLROnPlateau, ModelCheckpoint
- **Metrics**: Accuracy, Precision, Recall, F1-Score

## 📊 Model Performance

- **Architecture**: MobileNetV2 with custom classification head
- **Input Size**: 224x224x3
- **Training Strategy**: Transfer learning + Fine-tuning
- **Dataset Split**: Stratified 75% Train, 15% Validation, 10% Test (generated by baseline script)
- **Batch Size**: 24-32 (baseline configuration)
- **Epochs**: Two-stage baseline training with early stopping and LR reduction

## 💻 Usage

### Web Application

1. **Launch the app**: `cd SkinDisease && python -m streamlit run app.py`
2. **Upload an image**: Click "Browse files" and select a skin image
3. **Analyze**: Click "Analyze Image" button
4. **Review results**:


## 🩺 Doctor Consultation Recommendation System

The system provides doctor recommendations based on:

- **Detected Condition**: Predicted skin disease class
- **Specialist Type**: Mapped specialist guidance per condition
- **Preferred Language**: User-selected consultation language

Each recommendation includes:
- Doctor name
- Specialization
- Languages spoken
- Consultation phone/contact
- Consultation type (online/offline)

## 📝 Feedback Mechanism

Users can provide feedback on:
- Prediction accuracy (1-5 stars)
- Overall satisfaction (1-5 stars)
- Whether they would recommend the system
- Doctor consultation status
- Actual diagnosis (for accuracy tracking)
- Additional comments and suggestions

Feedback is stored in `results/user_feedback.csv` and can be viewed in the **Feedback History** page.

## 🔧 Troubleshooting

### Common Issues

**Application won't start:**
```bash
# Ensure virtual environment is activated
pip install -r SkinDisease/requirements.txt --upgrade

# Start the verified Streamlit entry point
cd SkinDisease
python -m streamlit run app.py
```

**Port 8501 is already in use:**
```bash
cd SkinDisease
python -m streamlit run app.py --server.port 8502
```

**Model not found error:**
- Check that model files exist in the root `models/` directory
- Ensure `class_names.txt` is present

**Low prediction confidence:**
- Use clear, well-lit photos
- Focus on the affected skin area
- Avoid blurry or dark images
